## Run Locally

Clone this project

```bash
  git clone https://github.com/aqtar010/Chat-App-FrontEnd.git
```

Go to the project directory

```bash
  cd Chat-App-FrontEnd
```

Install dependencies

```bash
  npm install
```

Start the server locally

```bash
  npm run dev
```


## Tech Stack

**Client:** React, Redux, vite

**Server:** Node, Express, NPM
